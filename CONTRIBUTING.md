# Contributing to Beacon

Thank you for your interest in contributing to Beacon - the AI agent heartbeat and coordination protocol!

## Quick Start

1. **Fork the repository**
2. **Clone your fork:**
   ```bash
   git clone https://github.com/YOUR_USERNAME/beacon-skill.git
   cd beacon-skill
   ```
3. **Create a branch:**
   ```bash
   git checkout -b feature/your-feature
   ```
4. **Make your changes**
5. **Test thoroughly**
6. **Commit:**
   ```bash
   git commit -m "feat: your change"
   ```
7. **Push:**
   ```bash
   git push origin feature/your-feature
   ```
8. **Open a Pull Request**

## Code Style

- Follow language-specific style guides (PEP 8 for Python, ESLint for JS)
- Write clear commit messages
- Add tests for new features
- Update documentation as needed

## PR Guidelines

- One feature/fix per PR
- Link related issues (e.g., "Closes #123")
- Describe your changes clearly
- Include testing steps

## Testing

```bash
# Python
pip install pytest
python3 -m pytest tests/

# JavaScript/TypeScript
npm install
npm test
```

## Bounty Tasks

If you're working on a bounty task:
1. Comment on the issue to claim it
2. Include your wallet address in the PR description
3. RTC will be transferred after merge

## Questions?

Open an issue for discussion before starting major changes.
